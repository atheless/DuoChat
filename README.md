# DuoChat

Chatting app using Django Channels and WebSockets.

### Installation
    git clone https://github.com/atheless/DuoChat.git
    cd DuoChat
    pipenv install
    pipenv shell
    pip install -r requirements.txt
    python manage.py runserver
